# Contributing

This project is Forenly AI Systems' submission to the Google Cloud Rapid Agent
Hackathon. We welcome contributors who are excited about the problem and able
to commit to hackathon-paced work.

## How to join

1. **Reach out** in the `Google_Cloud_Rapid_Agent_Hackathon` Rocket.Chat channel
   (or via DM to a team member listed in the README). Tell us:
   - Your background and what you'd bring
   - The hours you can commit
   - Which part of the project interests you
2. We'll grant **repo Read access** and share the current state of the work, plus
   questions where we'd like your perspective.
3. Once you've explored the repo and confirmed it's a fit, you can join the
   **planning conversation** — issues, design discussion, task triage.
4. From there, **code contributions** open up as collaboration develops.

We onboard step by step so new contributors land on work that genuinely fits
them, and so the team stays coordinated under hackathon time pressure.

## Once you're contributing code

- **Branches:** `feature/<short-name>`, `fix/<short-name>`, `docs/<short-name>`
- **Pull requests:** opened against `main`, squash-merged. Tag at least one team
  member for review.
- **Commits:** descriptive messages; reference issue numbers where applicable.

## Questions

If a team member pointed you here and you're not sure what to do next, just
message in Rocket.Chat — we'll guide you.
