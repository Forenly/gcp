How it works

A single agent, four MongoDB MCP tools, one grounded recommendation.

1

Describe the yard

Area, steepest slope, obstacles, boundary type and where power is — that's all the agent needs.

2

Agent queries the registry

Gemini calls MCP tools to find mowers within the yard's limits and pull plans from similar yards.

3

Get mower + plan

A primary pick, alternatives, and a deployment plan — dock placement, boundaries, zones and schedule — written back to MongoDB.
